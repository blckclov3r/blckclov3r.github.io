== Description ==
Kadence is a lightweight progressive theme optimized for fast loading and accessibility. Kadence has a complete header and footer builder so you can create the perfect header and footer for you site. Itâ€™s built for the modern era of WordPress and is tightly integrated with the WordPress core block editor. It also works great with any page builder and has custom support for Elementor, Beaver Builder and others.
