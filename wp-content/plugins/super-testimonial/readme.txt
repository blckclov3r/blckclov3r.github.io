<strong>How To Use Shortcodes? </strong>
basic Shortcode
<code>
[tpsscode]
</code>
<strong>Note: </strong>
Copy the following shortcode to the page, post or widget section to display the testimonial in the website.

<code>
[tpsscode themes="theme1"]
</code>
<strong>Note: </strong>You can use Four Different theme in your testimonials using shortcode parameters. theme shortcode parameters list: (themes1,themes2,themes3,themes4).

<code>
[tpsscode themes="theme1" category="your category id"]
</code>
<strong>Note: </strong>you can show multiple testimonials in the website using category id. Example category id:25

<code>
[tpsscode themes="theme1" category="your category id" order_by="date"]
</code>
<strong>Note: </strong>you can Sort your testimonials by date, menu_order & random.

<code>
[tpsscode themes="theme1" category="your category id" order_by="date" order="ASC"]
</code>
<strong>Note: </strong>you can Sort your testimonials by DESC/ASC.


== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see "S Testimonials" menu below the comment's menu at left side on WordPress dashboard and find "Shortcode"<br />
<br />

<strong>Using FTP</strong>
1.Extract tp-super-testimonials.zip to your computer.
2.Upload the tp-super-testimonials directory to your wp-content/plugins directory.
3.After activate plugin you will see "S Testimonials" menu at left side on WordPress dashboard and find "Shortcode"

<br />
<strong>How to use short-code?</strong><br />
When active & options setup done please paste [tpsscode] this shortcode into your theme any where you want to display this Testimonials.
